### Augmented Operators

Python's shorthand for combining arithmetic and bitwise operators with assignment operator (`=`)

```python [0|1|3-9|10-14]
a = b = c = d = e = f = g = h = i = j = k = l = 10

a += 3                # a = a + 3             13
b -= 3                # b = b - 3             7
c *= 3                # c = c * 3             30
d /= 3                # d = d / 3             3.333333333
e //= 3               # e = e // 3            3
f %= 3                # f = f % 3             1
g **= 3               # g = g ** 3            1000
h &= 3                # h = h & 3             2
i |= 3                # i = i | 3             11
j ^= 3                # i = i ^ 3             9
k <<= 3               # k = k << 3            80
l >>= 3               # l = l >> 3            1
```
