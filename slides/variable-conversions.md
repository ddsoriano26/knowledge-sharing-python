### Conversion of Variables

```python [0|2|3|4|5|6|7]
x = 10
type(10)                        # returns int
y = input("Input number")
type(y)                         # returns str
int(y)                          # returns int version of y
z = "five"
int(z)                          # returns ValueError
```

Other conversions:
- `str()`
- `float()`
- `bool()`
- `complex()`
